# HOW_TO_LIB_PRODUCT_INTRO_FIX.md

이 문서는 제품 소개 섹션의 레이아웃 로직과 이미지 비율 관리 규칙을 정리한 기술 서입니다.

## 1. 레이아웃 구조 (Flex Ratio)
제품 소개 열(`.product-row`)은 텍스트와 이미지의 비율을 **1 : 1.5**로 고정하여 시각적 일관성을 유지합니다.

- **텍스트 영역 (`.product-text`)**: `flex: 1`. 텍스트가 길어져도 이미지 영역을 침범하지 않도록 설정됨.
- **이미지 영역 (`.product-img`)**: `flex: 1.5`. 
- **이미지 비율**: `aspect-ratio: 16/9`가 적용되어 있어, 원본 이미지의 크기와 상관없이 항상 일정한 높이를 유지함.

## 2. 이미지 처리 규칙
- `object-fit: cover`: 이미지가 정해진 영역을 꽉 채우도록 설정되어 있어, 다양한 비율의 이미지를 업로드해도 레이아웃이 깨지지 않습니다.
- `border-radius: 8px`: 부드러운 모서리 처리가 기본으로 적용됩니다.

## 3. 유지보수 가이드
- 이미지 크기가 다시 달라 보인다면, `style.css`의 `flex` 속성과 `aspect-ratio` 설정을 확인하십시오.
- 모바일(768px 이하)에서는 자동으로 1열 세로 배치로 전환되며, 이미지 비율은 `4/3`으로 조정됩니다.
