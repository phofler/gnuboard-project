# HOW_TO_LIB_UNLOCK_RATIO.md

이 문서는 이미지 매니저의 크롭 비율 해제 기능에 대한 기술 서입니다.

## 1. 개요
기본적으로 섹션 관리자는 각 스킨의 권장 비율에 맞게 이미지를 자르도록 유도하지만, 사용자가 필요에 따라 자유로운 비율로 이미지를 편집할 수 있도록 기능을 제공합니다.

## 2. 구현 상세 (`adm/image_manager.php`)

### 1) 토글 버튼 (UI)
- `id="btn_toggle_ratio"`: 현재 상태에 따라 "비율 해제" 또는 "비율 고정"으로 텍스트가 바뀝니다.
- `id="ratio_info"`: 현재 적용된 비율 정보를 텍스트로 표시합니다.

### 2) 상태 제어 (JavaScript)
- `isRatioLocked`: 현재 비율 고정 여부를 저장하는 전역 플래그입니다. (기본값: `true`)
- `toggleAspectRatio()`:
    - `cropper.setAspectRatio(ratio)`를 호출하여 실시간으로 크롭 박스의 비율 제한을 변경합니다.
    - `ratio`가 `targetW / targetH`이면 고정, `NaN`이면 자유 비율이 됩니다.
- `initCrop(url)`: 새로운 이미지를 편집할 때마다 상태를 `true`(고정)로 초기화하여 일관성을 유지합니다.

## 3. 관리 지침
- 스킨마다 권장 비율이 다르므로 `targetW`와 `targetH` 변수를 통해 동적으로 비율을 계산합니다.
- 비율을 해제하면 사용자의 의도대로 잘리지만, 실제 웹사이트 레이아웃에서 이미지가 찌그러지거나 잘려 보일 수 있음을 유의해야 합니다. (스킨 CSS의 `object-fit: cover` 설정으로 보정됨)
