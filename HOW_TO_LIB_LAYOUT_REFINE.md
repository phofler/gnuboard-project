# HOW_TO_LIB_LAYOUT_REFINE.md

이 문서는 CEO 인사말 페이지의 디자인 고도화(v1, v2, v3)에 사용된 기술적 명세와 스타일 규칙을 정리한 문서입니다.

## 1. 이미지 반응형 및 배치 규칙 (`.con_img`)
- **컨테이너**: `max-width: 1000px`, `margin: 0 auto`로 중앙 정렬.
- **이격 관리 (v2)**: 상단 이미지와 하단 텍스트 사이의 간격은 `20px`로 유지하여 시각적 긴밀성을 확보함.
- **이미지 요소**: 
    - `max-width: 100% !important`, `height: auto !important`로 모든 기기 대응.
    - `border-radius: 12px`, `box-shadow`를 통해 프리미엄 느낌 부여.

## 2. 타이포그래피 및 레이아웃 고도화
- **폰트 크기 (v2)**: `clamp(1rem, 2.5vw, 1.5rem)` 적용. 화면 크기에 따라 유동적으로 변화하며, 고도화 이전에 비해 더욱 작고 세련된 느낌을 줌.
- **줄간격 (Line-height)**: `1.25`로 설정하여 여러 줄의 문구가 긴밀하게 연결되도록 함.
- **섹션 패딩 (v3)**: `.greeting-section`의 패딩을 `clamp(10px, 5vw, 40px) 0`으로 축소하여 상하 여백의 과도한 노출을 방지함.
- **간격 (Gap, v2)**: `.greeting-section`의 `gap`을 `clamp(10px, 2vw, 15px)`로 좁혀 전체적인 레이아웃의 짜임새를 최적화함.

## 3. 사용 가이드
- 웹에디터에서 이미지를 삽입할 때 `<div class="con_img">`로 감싸주면 자동으로 반응형 및 그림자 효과가 적용됩니다.
- 인사말 리드 문구는 `.greeting-lead` 클래스를 사용하여 통일된 디자인을 유지하십시오.
