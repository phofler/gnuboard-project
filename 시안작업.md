# 시안 작업 가이드 (Draft Layout Guide)

> **핵심 원칙 (Core Principle)**
> **"코딩(PHP) 전에, 반드시 눈으로 보는 시안(HTML)이 먼저다."**
> 복잡한 로직과 디자인을 동시에 잡으려 하지 마십시오. 완벽한 정적 페이지(Static Page)를 먼저 만들고, 그것을 승인받은 뒤에 스킨으로 이식합니다.

---

## 1. 시안 작업의 목적 (Purpose)
*   **Visual Verification**: PHP 변수나 루프(`for`) 없이 디자인 그 자체에만 집중하여 디테일(여백, 폰트, 호버 효과)을 완성합니다.
*   **Safety**: 라이브 서버의 데이터를 건드리지 않고 안전하게 레이아웃을 실험할 수 있습니다.
*   **Documentation**: 완성된 HTML 파일은 그 자체로 '살아있는 매뉴얼'이 됩니다.

## 2. 레이아웃 준수 원칙 (Sub-Layout Compliance) **[IMPORTANT]**
*   **모든 서브 페이지**(회사소개, 인사말, 갤러리 등)는 **기본 서브 시안(`sub_ko.html`)의 레이아웃 구조를 100% 준수**해야 합니다.
*   **Header, Footer, Hero Section**: 이 영역은 절대 변경하지 말고 그대로 복사하여 사용합니다.
*   **Content Area**: 오직 `<section class="content-section">` 내부의 콘텐츠만 페이지 성격에 맞게 변형합니다.
*   **일관성(Consistency)**: 사용자가 어떤 메뉴를 클릭하더라도 동일한 브랜드 경험(여백, 폰트, 헤더 높이)을 유지해야 합니다.

## 3. 작업 프로세스 (Process)

### Step 1: 껍데기 복사 (Shell Copy)
*   가장 신뢰할 수 있는 원본 파일(`sub_ko.html`)을 복사하여 새로운 프로토타입 파일(`gallery_prototype.html`)을 만듭니다.
*   이때 Header, Footer, Hero 섹션 등 **공통 레이아웃은 절대 건드리지 않고 유지**합니다.

### Step 2: 콘텐츠 이식 (Content Injection)
*   `sub_ko.html`의 `<section class="content-section">` 내부를 비우고, 새로 만들고자 하는 컴포넌트(예: 갤러리 그리드)를 작성합니다.
*   **CSS 작성 원칙**:
    *   `default.css`의 변수(`var(--c-brand)`, `var(--spacing-container)`)를 100% 활용합니다.
    *   새로운 스타일은 `<head>` 내 `<style>` 태그에 작성하여 격리합니다.

### Step 3: 인터랙션 구현 (Interaction)
*   필요한 기능(예: 클릭 시 레이어 팝업)은 바닐라 JS로 간단게 구현하여 '느낌'을 확인합니다.
*   복잡한 라이브러리보다는 가벼운 구현을 선호합니다.

### Step 4: 승인 및 이식 (Approval & Porting)
*   사용자에게 `http://localhost/.../gallery_prototype.html` 주소를 전달하여 컨펌을 받습니다.
*   컨펌이 완료되면, 해당 HTML/CSS를 그누보드 스킨 파일(`list.skin.php`, `style.css`)로 복사하여 PHP 코드를 입힙니다.

---

## 4. 예시: 갤러리 프로토타입 구조

```html
<!-- [Head] CSS 변수 및 폰트 로드 -->
<link rel="stylesheet" href="../../theme/corporate_light/css/default.css">

<!-- [Body] sub_ko.html과 동일한 헤더 구조 (변경 금지) -->
<nav>...</nav>
<div class="sub-hero">...</div>

<!-- [Content] 새로운 디자인 실험 (여기만 수정) -->
<section class="container content-section">
    <div class="gallery-grid">
        <!-- 카드 반복 -->
        <article class="gallery-card">
           ...
        </article>
    </div>
</section>
```
