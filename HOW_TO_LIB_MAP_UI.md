# HOW_TO_LIB_MAP_UI.md

이 문서는 지도 API 플러그인의 인포윈도우(InfoWindow) 커스텀 디자인 사양서입니다.

## 1. 개요
기본 제공되는 투박한 지도 말풍선을 사이트의 프리미엄 디자인 언어에 맞춰 커스텀 CSS로 재설계했습니다.

## 2. 구현 상세 (`plugin/map_api/skin.head.php`)

### 1) 스타일 구성
- **박스 모델**: 8px의 라운드 코너와 부드러운 그림자를 통해 공중에 떠 있는 듯한 입체감을 부여했습니다.
- **커스텀 화살표**: CSS `::after` 가상 요소를 사용하여 인포윈도우 하단에 말풍선 꼬리를 직접 구현했습니다.
- **타이포그래피**: 강조색(Corporate Blue)과 두꺼운 폰트 웨이트를 사용하여 브랜드명을 명확하게 인지하도록 했습니다.

### 2) 기술적 특징
- **강제 오버라이드**: 구글/네이버/카카오 지도의 기본 스타일을 덮어쓰기 위해 전용 클래스(`.map-api-info-window`)를 사용합니다.
- **반응형 대응**: `min-width`와 `padding`을 유동적으로 설정하여 모바일에서도 박스가 깨지지 않도록 했습니다.

## 3. 유지보수
- 박스 배경색을 바꾸려면 `.map-api-info-window`의 `background`와 `::after`의 `border-color`를 함께 수정해야 화살표 색상도 같이 바뀝니다.
