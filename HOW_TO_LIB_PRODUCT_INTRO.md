# HOW TO LIB PRODUCT INTRO METADATA

본 문서는 `product_intro` 섹션의 태그 및 소제목 렌더링 시스템의 로직을 설명합니다.

## 1. 아키텍처 개요
`product_intro` 스킨은 섹션 내 개별 아이템의 상세 메타데이터(태그, 소제목)를 시각적으로 강조하여 제품의 특성을 명확히 전달하도록 설계되었습니다.

## 2. 데이터 구조 및 렌더링 로직
1.  **태그(`mc_tag`)**: 
    *   아이템 제목(`h3`) 위에 위치하며, 강조색(`--mc-accent`) 배경의 배지 형태로 출력됩니다.
    *   데이터 존재 시에만 상자가 렌더링됩니다.
2.  **소제목(`mc_subtitle`)**: 
    *   태그와 제목 사이에 위치하며, 차분한 회색(#666)으로 부연 설명을 제공합니다.
3.  **반응형(Responsive)**: 
    *   모바일 환경에서는 텍스트 정렬을 중앙으로 변경하고, 가독성을 위해 간격을 재조정합니다.

## 3. 주요 CSS 클래스
*   `.product-tag`: 태그 배지 스타일 (그림자, 둥근 모서리 적용)
*   `.item-subtitle`: 개별 아이템 소제목 스타일
*   `.btn-detail`: '자세히 보기' 버튼 (호버 시 이동 효과)

## 4. 커스텀 가이드
태그의 색상을 변경하려면 `skin.head.php` 또는 관리자 페이지의 섹션 설정에서 `$ms_accent_color`를 조정하십시오. 이 값은 CSS 변수 `--mc-accent`를 통해 스타일시트에 반영됩니다.
