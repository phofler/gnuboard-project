# 토스페이먼츠(Toss Payments) 연동 가이드

## 1. 개요
토스페이먼츠는 최신 방식의 PG사로, 웹/모바일 통합 연동이 간편하며 별도의 모바일 패치가 거의 필요 없습니다.

## 2. 키 값 설정
`toss_config.php` 파일에 있는 두 가지 키 값을 설정합니다.
- **Client Key**: 결제창(JavaScript) 호출 시 사용
- **Secret Key**: 결제 승인(Server-to-Server) 처리 시 사용

## 3. 그누보드/영카트 설정
1. 관리자 페이지 > 쇼핑몰관리 > 쇼핑몰설정 > **결제설정** 탭 이동
2. 결제대행사를 **[토스페이먼츠]**로 선택
3. 위에서 확인한 `Client Key`와 `Secret Key`를 각각 입력

## 4. 자주 묻는 질문 (FAQ)
Q. 모바일에서 결제창이 안 뜹니다.
A. 토스페이먼츠는 `https` (보안연결) 환경에서만 정상 작동하는 경우가 많습니다. 로컬(`localhost`)에서는 되지만 실서버에서는 SSL 인증서가 필수입니다.

Q. 결제 승인이 안 됩니다.
A. `Secret Key`가 올바른지 확인해 주세요. 클라이언트 키만 넣고 시크릿 키를 누락하는 경우가 많습니다.
