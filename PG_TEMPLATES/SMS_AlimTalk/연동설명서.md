# SMS / 알림톡 연동 가이드 (Priority #4)

## 1. 서비스 개요
- **그누보드 기본**: 아이코드(iCode)가 내장되어 있어 설정만 하면 바로 나갑니다.
- **외부 업체(알리고, 가비아 등)**: API 연동 개발이 필요합니다. (비용 절감 목적)

## 2. 아이코드 설정 (가장 쉬움)
관리자 > 환경설정 > 기본환경설정 > **SMS 설정**
- 아이디/비밀번호 입력만으로 끝.

## 3. 카카오 알림톡 (고급)
알림톡은 SMS보다 저렴하고 신뢰도가 높습니다.
1. '알리고'나 '비즈톡' 등 중계사 가입
2. 카카오 채널 개설 및 비즈니스 인증
3. **템플릿 등록**: 보낼 메시지 내용을 미리 카카오에 검수받아야 함 (가장 중요)
4. 코드 연동: `lib/alimtalk.lib.php` (별도 개발 필요) 형태의 라이브러리 제작

## 4. 팁
- 쇼핑몰(#2)과 연동 시: 주문완료, 입금확인, 배송출발 시점에 자동 발송 로직 추가 (`/shop/orderformupdate.php` 등 훅 사용)
