# KG이니시스 연동 가이드

## 1. 설정 파일 적용
가장 간편한 방법은 영카트 관리자 페이지 설정을 따르지만, 코드로 강제 고정하고 싶을 때 이 파일을 사용합니다.

1. `inicis_config.php` 파일을 엽니다.
2. `INIPayTest` 부분을 실제 상점 ID로 변경합니다.
3. 사인키(SignKey)를 발급받은 실제 키로 변경합니다.
4. 이 내용을 `/shop/settle_kg_inicis.inc.php` 등의 설정 파일 상단에 include 하거나, 해당 변수 값을 덮어씌웁니다.

## 2. 모바일 결제 오류 패치
모바일에서 결제창 호출 시 'Order ID' 불일치 오류가 날 경우:
- 파일: `/mobile/shop/orderform.php`
- 패치 내용: 세션 도메인 설정 확인 (`set_session('ss_order_id', $od_id);` 부분 체크)

## 3. 주의사항
- **테스트 결제**는 반드시 'INIpayTest'로 진행해야 합니다.
- 실제 서비스 전환 시, 이니시스 관리자에 등록된 IP와 서버 IP가 일치해야 합니다.
