# 🗞️ 보드 최근글 스킨 관리 표준 (Latest Skin Standards)

> **"메인 페이지의 얼굴, 최신글 위젯의 기술 표준입니다."**
> 본 문서는 테마의 **`default.css`** 디자인 토큰을 엄격히 준수합니다.

### 0.1 디자인 시스템 참조 (Design System Reference)
최신글 스킨 제작 시 **현재 활성화된 테마(Active Theme)**의 **`css/default.css`** 전역 변수를 활용하십시오.

*   **Colors**: `var(--color-title-main)`(메인제목), `var(--color-primary)`(강조), `var(--color-text-body)`(본문)
*   **Fonts**: `var(--font-header)`(영문), `var(--font-body)`(한글)
*   **Spacing**: `var(--spacing-section)`(섹션간격), `var(--container-padding)`(반응형 여백)

---

## 1. 개요 (Overview)
*   **용도**: 메인 페이지나 사이드바에 노출되는 최근 게시물 위젯.
*   **위치**: `/theme/corporate_light/skin/latest/` (반드시 테마 폴더 내에 위치)
*   **식별자**:
    *   폴더명: `skin_타입명` (예: `skin_webzine`, `skin_gallery`)
    *   DB ID: `corporate_main_news` 등.

## 2. 개발 표준 (Development Standards)

### 2.1 표준 변수
*   **`$options`**: 위젯 설정값 (타이틀, 노출 개수 등)을 넘겨받아 동적으로 렌더링해야 합니다.
    ```php
    // 예시: 옵션값 활용
    $title = isset($options['title']) ? $options['title'] : 'Latest News';
    ```

### 2.2 디자인 원칙 (Design Principles)
*   **No H1/H2**: 위젯 내부 타이틀은 `H3` 이하 또는 `div.widget-title`을 사용하십시오. (SEO 구조 보호)
*   **More Button**: 더보기 버튼 클릭 시 해당 게시판 목록(`$bo_href`)으로 이동해야 합니다.
*   **Responsive**: 모바일에서는 1단, 데스크탑에서는 그리드/리스트 등 반응형 레이아웃을 준수해야 합니다.

---

## 3. 주요 스킨 목록
*   **`skin_webzine`**: 텍스트 중심의 뉴스 리스트.
*   **`skin_gallery`**: 이미지 중심의 포트폴리오 썸네일.
*   **`skin_slider`**: 메인 비주얼용 슬라이더 (Swiper.js 활용).
