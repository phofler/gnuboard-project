# 그누보드 게시판 글쓰기(Write) 폼 구조 분석

본 문서는 그누보드5의 표준 `write.skin.php` 및 커스텀 테마(`corporate_light`)의 글쓰기 페이지 구조를 분석한 내용입니다. 향후 HTML 프로토타입을 실제 스킨으로 변환할 때 참고자료로 활용됩니다.

## 1. 폼(Form) 기본 구조
*   **Action URL**: `$action_url` (주로 `write_update.php`로 전송)
*   **Enctype**: `multipart/form-data` (파일 첨부 필수)
*   **Hidden Fields (필수)**:
    *   `w`: 작성 모드 (`''`: 새글, `'u'`: 수정, `'r'`: 답변)
    *   `bo_table`: 게시판 테이블 ID
    *   `wr_id`: 게시물 ID (수정/답변 시)
    *   `uid`: 유니크 ID (`get_uniqid()`)
    *   `sca`, `sfl`, `stx`, `page`: 리스트 검색 및 페이징 상태 유지용

## 2. 주요 입력 필드 (Input Fields)

### A. 카테고리 (Category)
*   **표준**: `name="ca_name"` (Select 박스 또는 Input)
*   **테마 커스텀 (Cascading)**:
    *   이 테마는 `ca_name`을 `display:none`으로 숨기고, 자바스크립트로 3단(대/중/소) 카테고리를 동적 생성함.
    *   사용자가 선택한 값은 최종적으로 `ca_name`과 여분필드 `wr_1` (카테고리 코드)에 저장됨.

### B. 옵션 (Options)
*   `notice`: 공지사항 (관리자 전용)
*   `html`: HTML 사용 여부 (`html1`, `html2`)
*   `secret`: 비밀글 (`secret`)
*   `mail`: 답변 메일 받기

### C. 기본 정보 (비회원/Guest)
*   `wr_name`: 이름 (필수)
*   `wr_password`: 비밀번호 (필수)
*   `wr_email`: 이메일
*   `wr_homepage`: 홈페이지

### D. 본문 내용 (Content)
*   `wr_subject`: 제목 (필수)
*   `wr_content`: 내용 (에디터 또는 텍스트에어리어)
    *   **Editor**: `$is_dhtml_editor` 값에 따라 웹 에디터 로드.
    *   **Validation**: 자바스크립트 `fwrite_submit` 함수에서 금지단어, 글자수 제한 등을 체크.

### E. 링크 및 파일 (Links & Files)
*   `wr_link[i]`: 링크 입력 (보통 1~2개)
*   `bf_file[]`: 파일 첨부 (`input type="file"`)
*   `bf_content[]`: 파일 설명 (이미지 alt 태그나 다운로드 설명용)

## 3. 자바스크립트 로직 (JS Logic)
*   **`fwrite_submit(f)`**: 폼 전송 시 실행되는 핵심 함수.
    1.  **에디터 내용 동기화**: `editor_js` 실행.
    2.  **금지단어 체크**: `ajax.filter.php`를 통해 비동기 검사.
    3.  **글자수 제한**: `check_byte` 함수로 최소/최대 글자수 검증.
    4.  **CAPTCHA**: 자동등록방지 코드 검증.
    5.  **이중 전송 방지**: 전송 버튼 비활성화 (`disabled`).

## 4. 테마 스킨 특이사항 (Reference: `gallery_editorial`)
*   **`me_code` 유지**: 페이지 이동 시 메뉴 상태 유지를 위해 GET 파라미터 `me_code`를 폼 Action과 취소 버튼 링크에 포함시킴.
*   **취소 버튼 앵커**: 취소 시 단순히 목록으로 가는 것이 아니라 `#bo_cate` 위치로 이동하여 스크롤 위치를 보정함.
