# 메인 컨텐츠 관리 스킨 제작 가이드 (Main Content Skin Guide)

> **Core Standard**:
> 모든 스타일은 **현재 활성화된 테마(Active Theme)**의 `css/default.css` 토큰을 상속받아야 합니다.
> (예: `theme/{your_theme}/css/default.css`)

## 1. 개요 (Overview)
`main_content_manager` 플러그인은 메인 페이지의 핵심 섹션(Services, Portfolio, Features 등)을 반복 가능한 목록 형태로 관리하는 도구입니다.

### 🎨 Design System Reference
필수적으로 참조해야 할 테마 변수입니다.

| 변수명 | 설명 | 비고 |
| :--- | :--- | :--- |
| `--mc-section-padding` | 섹션 수직 여백 | 기본 100px |
| `--mc-item-gutter` | 아이템 간 간격 | 기본 50px |
| `--mc-title-size` | 섹션 메인 타이틀 크기 | 3rem |
| `--mc-transition` | 애니메이션 곡선 | cubic-bezier |

## 2. 스킨 구조 (Skin Structure)
모든 스킨은 아래의 표준 클래스 구조를 따라야 합니다.

```php
<section class="mc-skin-wrapper skin_theme_name">
    <!-- Container: 1360px -->
    <div class="skin-container">
        <!-- Header -->
        <div class="mc-header">
            <h2 class="mc-title"><?php echo $ms['ms_title']; ?></h2>
            <?php if($ms['ms_desc']) { ?>
                <p class="mc-desc"><?php echo $ms['ms_desc']; ?></p>
            <?php } ?>
        </div>
        
        <!-- Grid/List -->
        <div class="mc-list-wrap">
            <?php foreach($items as $item) { ?>
                <div class="mc-item">
                    <!-- Item Implementation -->
                </div>
            <?php } ?>
        </div>
    </div>
</section>
```

## 3. 핵심 규칙 (Core Rules)
1.  **반응형 그리드**: CSS Grid를 사용하여 PC/Mobile 레이아웃을 제어합니다.
2.  **데이터 바인딩**: `$items` 배열을 순회하며 `$item['mc_title']`, `$item['mc_desc']` 등을 출력합니다.

## 4. 단독 레이아웃형 (Standalone Layout Type)
리스트 반복이 없는 **고정형 컨텐츠**(철학, 지도, 문의폼 등)를 구현하는 방식입니다.
**'회사소개 관리'**에서 컨텐츠를 만들고, **'메인 컨텐츠 관리'**에서 불러오는 **2단계 구조(Bridge Pattern)**를 따릅니다.

### 🚀 제작 및 연결 절차
1.  **소스 생성 (Source)**:
    *   위치: **[관리자] 회사소개 관리 > 회사소개 추가**
    *   분류: **`메인 섹션 전용 (Main Page)`** 그룹 선택.
    *   스킨: `main_manifesto`, `main_map` 등 목적에 맞는 스킨 선택 후 저장.

2.  **출력 연결 (Connect)**:
    *   위치: **[관리자] 메인 컨텐츠 관리 > 섹션 추가**
    *   스킨: **`메인 지도 (Main Map)`** 또는 **`Style Philosophy`** 등 해당 연동 스킨 선택.
    *   설정: **`회사소개 컨텐츠 선택`** 드롭다운에서 1번에서 만든 항목을 선택.

### 4.1 적용 예시 (Use Cases)
*   **Main Map**: 지도 API와 오시는 길 정보를 담은 `company_intro` 데이터를 메인 섹션으로 불러옵니다.
*   **Manifesto**: 정적인 기업 철학 텍스트를 메인 섹션으로 불러옵니다.

### 💻 코드 구현 예시
단독형 스킨은 루프(`foreach`) 없이, 연결된 회사소개 ID를 이용해 내용을 가져옵니다.

```php
<section class="mc-skin-wrapper">
    <!-- ID로 회사소개 내용 가져오기 -->
    <?php
    // $ms['ms_content_id']에 연결된 회사소개 ID가 저장됨
    echo display_company_content($ms['ms_content_id']); 
    ?>
</section>
```
